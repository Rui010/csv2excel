How To USE

CSVファイルをドラッグアンドドロップ、または、ファイル選択よりアップロードしてください。

EXCELファイルに変換され、ダウンロードフォルダに格納されます。

メリット

  •  無料で使用できるWebサービスです
  •  5MBまでのCSVファイルに対応
  •  外部サーバではなく、ブラウザの処理
  •  流出リスク0で社外秘ファイルも変換可能
